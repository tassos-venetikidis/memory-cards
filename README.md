# Memory Cards

A simple web application that facilitates the creation of flash cards that help with studying and memorizing things. Just press the Add New Card to start making your own cards that include a question and an answer. Cards are then displayed with the question side up and when clicked on, rotate to reveal the answer on the back. Created cards are saved to localStorage, so that you have access to them between sessions. To traverse your memory card deck use the navigation arrows just below the cards. Sliding card effects were done using plain CSS and functionality is supported by vanilla JavaScript.

Made with:

- HTML & CSS, for the markup and styling.
- Vanilla JS for the functionality.
- The localStorage web API for state persistence.
